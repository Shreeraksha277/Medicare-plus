# 🏥 MediCare Plus

MediCare Plus is a full-stack healthcare appointment booking web application designed to streamline the interaction between patients, doctors, and hospitals through a simple and user-friendly digital platform.

The application enables patients to register, explore available healthcare services, and book appointments efficiently while maintaining an organized appointment workflow.

---

## ✨ Key Features

- Patient Registration System
- Unique Patient ID Generation
- Doctor Specialization Selection
- Hospital and Doctor Listings
- Consultation Fee and Timing Display
- Appointment Booking System
- Booking Confirmation and Receipt Generation
- Responsive User Interface

---

## 💻 Technology Stack

### Frontend
- React
- TypeScript
- Tailwind CSS

### Backend
- Express.js
- TypeScript
- Node.js

---

## 🔄 Application Workflow

1. Patient registers with personal details  
2. System generates a unique Patient ID  
3. Patient selects the required doctor specialization  
4. Available hospitals and doctors are displayed  
5. Patient books an appointment  
6. System generates appointment confirmation
